.. Title

codeaffen.phpipam.vlan -- Manage vlans
++++++++++++++++++++++++++++++++++++++

.. Collection note

.. note::
    This plugin is part of the `codeaffen.phpipam collection <https://galaxy.ansible.com/codeaffen/phpipam>`_ (version 1.5.0).

    You might already have this collection installed if you are using the ``ansible`` package.
    It is not included in ``ansible-core``.
    To check whether it is installed, run :code:`ansible-galaxy collection list`.

    To install it, use: :code:`ansible-galaxy collection install codeaffen.phpipam`.

    To use it in a playbook, specify: :code:`codeaffen.phpipam.vlan`.

Requirements

- inflection
- ipaddress
- phpypam>=1.0.0

Examples
--------

.. code-block:: yaml+jinja

    
    - name: "Create vlan"
      codeaffen.phpipam.vlan:
        username: "admin"
        password: "s3cr3t"
        server_url: "https://ipam.example.com"
        name: "my vlan"
        vlan_id: 1337
        state: present

    - name: "Remove vlan"
      codeaffen.phpipam.vlan:
        username: "admin"
        password: "s3cr3t"
        server_url: "https://ipam.example.com"
        name: "my vlan"
        state: absent
